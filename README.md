# Jack Apple Team Website

A modern static website made for GitHub Pages.

## Upload to GitHub

1. Open your GitHub repository.
2. Click **Add file** → **Upload files**.
3. Upload:
   - `index.html`
   - `style.css`
   - `script.js`
4. Commit the files.
5. Open **Settings** → **Pages**.
6. Select **Deploy from a branch**.
7. Choose the `main` branch and `/ (root)`.
8. Save.

Your custom domain should then show this website once GitHub Pages finishes deploying.

## Edit the website

- Main text and sections: `index.html`
- Colors and design: `style.css`
- Mobile menu and animations: `script.js`

The site does not need Node.js, npm, or a build command.
